# Hacker News Post

## Title
**Show HN: VectorLens – Open-source UI for browsing and searching ChromaDB collections**

## URL
https://github.com/jarvismaia97/vectorlens

## Body (text, if self-post)

VectorLens is a React dashboard for ChromaDB that lets you browse collections, run semantic searches with similarity scores, visualize document timelines, and explore memory graphs — all without writing code.

Features: collection browser, natural language search, timeline view, force-directed relationship graph, duplicate detection, dark/light mode.

Stack: React 19, TypeScript, Tailwind CSS 4, Vite 7, FastAPI proxy. Ships with Docker Compose.

GitHub: https://github.com/jarvismaia97/vectorlens
